# SVORA

**SVORA** is a premium, AI-driven wellness and fitness platform that bridges the gap between minimalist luxury aesthetics and advanced biometric intelligence. It delivers a "Boutique Wellness" experience through a sophisticated "Cyber-Zen" interface, guiding users from somatic profiling to the generation of high-fidelity, periodized training protocols.

## // CORE PHILOSOPHY

- **Somatic Intelligence:** High-precision body analysis via computer vision or manual profiling.
- **Aesthetic Precision:** Targeted hypertrophy, lean mass, and power blueprints tailored to your specific somatic frame.
- **Cyber-Zen Aesthetic:** A "Boutique Wellness" digital environment—minimalist, immersive, and high-tech.
- **Protocol Synthesis:** Structured 12-week transformation journeys with metabolic strategies and exercise intelligence.
- **Environmental Adaptation:** Seamlessly switches between home, gym, and lifestyle-specific contexts (fasting, scheduling).

## // THE EXPERIENCE (Cyber-Zen Luxe)

- **Palette:** Jet Black (`#000000`), Gold Leaf (`#D4AF37`), Mercury Silver (`#E5E5E5`), and Emerald Glow (`#10B981`).
- **Typography:** `Playfair Display` (Serif/Italic) for elegance, `Inter` (Sans) for clarity, and `JetBrains Mono` for technical precision.
- **Interaction:** Kinetic buttons, magnetic custom cursors, bento-luxe grids, and spectral light leak atmospheres.

## // TECH STACK (Next-Gen)

- **Frontend:** Next.js 15+ (App Router, React 19)
- **Styling:** Tailwind CSS v4 (High-Definition Palette, @theme variable system)
- **Animations:** Framer Motion (Ultra-smooth kinetic transitions, parallax layers)
- **AI Engine:** OpenRouter SDK (Trinity Large & Gemini 2.0 Flash Integration)
- **Biometrics:** MediaPipe (Pose estimation for body analysis)
- **Database:** MongoDB Atlas (Persistent user profiles and training history)
- **Icons:** Lucide React

## // OPERATIONAL STARTUP

### 1. Repository Calibration
```bash
git clone <repo-url>
cd fitness
npm install
```

### 2. Neural Link (Environment)
Create a `.env.local` file in the root directory:
```env
OPENROUTER_API_KEY=your_key_here
MONGODB_URI=your_uri_here
JWT_SECRET=your_secret_here
```

### 3. Initialize System
```bash
npm run dev
```

## // PROJECT STRUCTURE

- `src/app/uplink`: Biometric intake and somatic profiling.
- `src/app/protocol`: AI-generated training blueprints.
- `src/app/journal`: Daily progress logging and performance tracking.
- `src/app/api`: Microservices for body analysis and protocol synthesis.
- `src/components`: Reusable "Boutique" UI units (`Preloader`, `CustomCursor`, `DecryptedText`).
